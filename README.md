# vkGroupBot

Bot for simplified group management Vkontakte.


# О боте
Бот может:
  - Автоматически опубликовывать предложенные записи.
  - Публикация/удаление постов через сообщения с сообществом.
  - Отвечать пользователю по определенным фразам.

vkGroupBot- это инструмент, созданный для упрощенного управления сообществом Вконтакте. 

### Технологии

vkGroupBot использует следующие технологии:

*   vk-io - SDK
*  node-vk-bot-api -  SDK
* [node.js] - evented I/O для серверной части


### Установка

Установите  [Node.js](https://nodejs.org/) для запуска бота.

Настройте config.js под себя.

Установите зависемости и запустите сервер.

```sh
$ cd vkGroupBot
$ npm install -d
$ node bot
```
